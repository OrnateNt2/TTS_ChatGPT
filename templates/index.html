Преобразование текста в речь

{% with messages = get_flashed_messages() %} {% if messages %}
{% for message in messages %}
{{ message }}
{% endfor %}
{% endif %} {% endwith %}
Различия моделей:
  • tts-1: Базовая модель преобразования текста в речь с быстрой обработкой, подходит для менее требовательных приложений.
  • tts-1-hd: Модель высокого качества, обеспечивает более естественное звучание, но может быть немного медленнее.
  • whisper-1: Модель для распознавания речи (speech-to-text), а не для генерации аудио.
Ограничения по символам:

Рекомендуется не превышать 1000–2000 символов за один запрос. Для больших текстов разбивайте его на части.
